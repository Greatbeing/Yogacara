# Yogacara 涌现机制与觉醒判断优化设计

## 一、问题分析

### 1.1 当前涌现机制的问题

```python
# 当前机制（emergence.py）
MIN_SEEDS_FOR_EMERGENCE = 3      # 固定阈值
SYNERGY_THRESHOLD = 0.6          # 固定阈值
STRENGTH_THRESHOLD = 0.7          # 固定阈值
```

**核心问题**：
1. **阈值静态化**：三个核心阈值都是硬编码的固定值，缺乏对系统动态状态的感知
2. **缺乏相变特征**：真正的涌现具有相变特征（Phase Transition），当前机制只是简单的计数与加权
3. **协同计算浅层化**：只计算类型多样性和协同矩阵，缺乏对深度协同模式的识别
4. **无临界态检测**：复杂系统中，接近临界点时会出现特征波动放大现象，当前机制无法检测

### 1.2 当前觉醒判断的问题

```python
# 当前机制（awakening.py）
LEVEL_THRESHOLDS = {
    "L2": {"wisdom_pct": 10, "compassion_pct": 5, "emergence": 1},
    "L3": {"wisdom_pct": 20, "compassion_pct": 10, "emergence": 3},
    ...
}
```

**核心问题**：
1. **依赖简单计数**：只统计涌现次数（emergence_count），忽视每次涌现的质量差异
2. **缺乏连续性感知**：两次高强度涌现 vs 多次低强度涌现，当前机制等权对待
3. **无势能积累概念**：觉醒应该像水位上升，有势能积累过程
4. **缺乏预警机制**：无法预知即将发生的跃迁

## 二、理论基础：复杂性科学对涌现的启发

### 2.1 相变理论（Phase Transition）

真实涌现系统具有以下相变特征：

| 特征 | 描述 | 应用于 Yogacara |
|------|------|-----------------|
| **临界态** | 系统在有序/无序边界时最敏感 | 种子系统接近临界态时最易涌现 |
| **幂律分布** | 少数种子驱动大量效应 | 关键种子主导涌现强度 |
| **临界波动** | 临界点附近波动放大 | 涌现前的"躁动期" |
| **不可预测性** | 接近临界时系统行为不可预测 | 涌现触发时机具有随机性 |

### 2.2 重整化群思想（Renormalization Group）

- **尺度不变性**：不同层次的种子应该具有自相似性
- **有效场论**：可以用粗粒化的"有效参数"描述复杂系统
- **固定点**：系统在临界点附近趋于某种"固定模式"

## 三、优化方案

### 3.1 涌现机制优化：引入相变感知

#### 3.1.1 临界阈值动态化

将固定阈值改为基于系统状态的动态阈值：

```python
class DynamicThreshold:
    """
    动态临界阈值计算
    
    核心思想：
    - 系统熵低时（有序），阈值适当降低（容易涌现）
    - 系统熵高时（混乱），阈值适当提高（难以涌现）
    - 临界态附近波动放大
    """
    
    def __init__(self, base_threshold: float, sensitivity: float = 0.1):
        self.base = base_threshold
        self.sensitivity = sensitivity
    
    def calculate(self, system_entropy: float, seed_correlation: float) -> float:
        """
        计算动态阈值
        
        Args:
            system_entropy: 系统熵 (0=完全有序, 1=完全混乱)
            seed_correlation: 种子间关联度 (0-1)
        
        Returns:
            调整后的阈值
        """
        # 熵调整：熵高时提高阈值，熵低时降低阈值
        entropy_adjustment = (0.5 - system_entropy) * self.sensitivity
        
        # 关联度调整：高关联度时降低阈值（接近相变）
        correlation_adjustment = seed_correlation * 0.1
        
        return max(0.1, min(0.9, self.base + entropy_adjustment - correlation_adjustment))
```

#### 3.1.2 幂律特征检测

```python
class PowerLawDetector:
    """
    检测种子系统是否呈现幂律特征
    
    幂律分布 P(x) ∝ x^(-α) 是复杂系统自组织的标志
    """
    
    def calculate_alpha(self, values: List[float]) -> Optional[float]:
        """
        使用最大似然估计计算幂律指数 α
        
        Returns:
            α 值（通常在 2-3 之间表示典型的复杂系统）
            None 表示不满足幂律
        """
        if len(values) < 10:
            return None
        
        # 简化的MLE估计
        x_min = min(values)
        n = len(values)
        
        # α = 1 + n / Σ ln(x / x_min)
        sum_log = sum(math.log(x / x_min) for x in values if x > x_min)
        if sum_log == 0:
            return None
        
        alpha = 1 + n / sum_log
        
        # α 合理性检查（实际系统中通常在 1.5-3.5）
        if 1.5 <= alpha <= 3.5:
            return alpha
        return None
    
    def detect_emergence_likelihood(self, seeds: List[Seed]) -> float:
        """
        基于幂律特征检测涌现可能性
        
        Returns:
            0.0-1.0 的涌现可能性指数
        """
        # 提取种子纯度序列
        purities = [s.purity for s in seeds]
        
        alpha = self.calculate_alpha(purities)
        if alpha is None:
            return 0.3  # 无法判断，返回中等可能性
        
        # α 越接近 2.5，涌现可能性越高（典型临界态）
        deviation = abs(alpha - 2.5)
        likelihood = max(0, 1 - deviation / 1.5)
        
        return likelihood
```

#### 3.1.3 临界波动检测

```python
class CriticalFluctuationDetector:
    """
    临界波动检测
    
    在相变临界点附近，系统的宏观量会出现特征性波动：
    - 波动幅度放大
    - 波动持续时间延长
    - 不同自由度间关联增强
    """
    
    def __init__(self, window_size: int = 10):
        self.window_size = window_size
        self.history: List[Dict[str, float]] = []
    
    def record_state(self, purity_std: float, synergy_var: float, correlation: float):
        """记录系统状态"""
        self.history.append({
            "purity_std": purity_std,
            "synergy_var": synergy_var,
            "correlation": correlation
        })
        
        # 保持滑动窗口
        if len(self.history) > self.window_size * 2:
            self.history.pop(0)
    
    def detect_critical_fluctuation(self) -> Tuple[bool, float]:
        """
        检测是否处于临界波动状态
        
        Returns:
            (是否临界, 临界程度 0-1)
        """
        if len(self.history) < self.window_size:
            return False, 0.0
        
        # 分析最近 window_size 个状态
        recent = self.history[-self.window_size:]
        
        # 计算波动指标
        purity_volatility = self._calculate_volatility([s["purity_std"] for s in recent])
        synergy_volatility = self._calculate_volatility([s["synergy_var"] for s in recent])
        correlation_trend = self._calculate_trend([s["correlation"] for s in recent])
        
        # 临界判断：高波动 + 关联增强
        is_critical = (
            purity_volatility > 0.3 and 
            synergy_volatility > 0.2 and 
            correlation_trend > 0
        )
        
        # 计算临界程度
        criticality = min(1.0, (purity_volatility + synergy_volatility) / 2)
        
        return is_critical, criticality
    
    def _calculate_volatility(self, values: List[float]) -> float:
        """计算波动性（标准差/均值）"""
        if not values or sum(values) == 0:
            return 0.0
        mean = sum(values) / len(values)
        variance = sum((v - mean) ** 2 for v in values) / len(values)
        return math.sqrt(variance) / (mean + 0.001)
    
    def _calculate_trend(self, values: List[float]) -> float:
        """计算趋势（正=上升）"""
        if len(values) < 2:
            return 0.0
        # 简单线性回归斜率
        n = len(values)
        x_mean = (n - 1) / 2
        y_mean = sum(values) / n
        
        numerator = sum((i - x_mean) * (v - y_mean) for i, v in enumerate(values))
        denominator = sum((i - x_mean) ** 2 for i in range(n))
        
        if denominator == 0:
            return 0.0
        return numerator / denominator
```

### 3.2 觉醒判断优化：从计数到质量

#### 3.2.1 觉醒能量（Awakening Energy）

引入"觉醒能量"替代简单的涌现计数：

```python
@dataclass
class AwakeningEnergy:
    """
    觉醒能量：综合衡量觉醒质量的核心指标
    
    不再依赖简单的涌现次数，而是考虑：
    - 涌现强度（每次涌现的 strength）
    - 涌现连续性（近期涌现的密度）
    - 涌现多样性（不同类型涌现的分布）
    """
    
    # 衰减参数
    HALF_LIFE_DAYS = 7  # 能量半衰期
    
    @staticmethod
    def calculate(
        emergence_history: List[Emergence],
        days_since_emergence: List[int]
    ) -> float:
        """
        计算当前觉醒能量
        
        Args:
            emergence_history: 涌现历史
            days_since_emergence: 每次涌现距今天数
        
        Returns:
            觉醒能量值 (0-1)
        """
        if not emergence_history:
            return 0.0
        
        # 1. 强度贡献（加权求和）
        intensity_contribution = sum(
            e.strength * AwakeningEnergy._time_decay(d)
            for e, d in zip(emergence_history, days_since_emergence)
        )
        
        # 2. 连续性加成（近期多次涌现）
        recent_count = sum(1 for d in days_since_emergence if d <= 3)
        continuity_bonus = min(0.2, recent_count * 0.05)
        
        # 3. 多样性加成（不同类型涌现）
        emergence_types = set(e.emergence_type for e in emergence_history)
        diversity_bonus = len(emergence_types) * 0.03
        
        # 4. 归一化
        base = min(1.0, intensity_contribution)
        total = base + continuity_bonus + diversity_bonus
        
        return min(1.0, total)
    
    @staticmethod
    def _time_decay(days: int, half_life: float = None) -> float:
        """时间衰减因子（指数衰减）"""
        if half_life is None:
            half_life = AwakeningEnergy.HALF_LIFE_DAYS
        return math.exp(-0.693 * days / half_life)
```

#### 3.2.2 觉醒势能（Awakening Potential）

```python
@dataclass
class AwakeningPotential:
    """
    觉醒势能：衡量距离下一觉醒等级的可能性
    
    考虑因素：
    - 智慧/慈悲种子的纯度分布
    - 种子激活频率（vasana）
    - 临界态指标
    """
    
    @staticmethod
    def calculate(
        wisdom_seeds: List[Seed],
        compassion_seeds: List[Seed],
        criticality: float = 0.0
    ) -> float:
        """
        计算觉醒势能
        
        Returns:
            势能值 (0-1)，越高越接近觉醒跃迁
        """
        # 1. 纯度分布势能
        wisdom_potential = AwakeningPotential._seed_distribution_potential(wisdom_seeds)
        compassion_potential = AwakeningPotential._seed_distribution_potential(compassion_seeds)
        
        # 2. 激活势能（高频激活的种子贡献更大）
        wisdom_activation = sum(s.vasana * s.purity for s in wisdom_seeds)
        compassion_activation = sum(s.vasana * s.purity for s in compassion_seeds)
        
        # 3. 归一化组合
        purity_component = (wisdom_potential + compassion_potential) / 2
        activation_component = (wisdom_activation + compassion_activation) / 100
        
        # 4. 临界加成
        criticality_bonus = criticality * 0.2
        
        potential = purity_component * 0.5 + activation_component * 0.3 + criticality_bonus
        
        return min(1.0, potential)
    
    @staticmethod
    def _seed_distribution_potential(seeds: List[Seed]) -> float:
        """
        计算种子分布势能
        
        核心理念：高纯度种子占比越高，势能越大
        同时考虑种子数量的边际效用递减
        """
        if not seeds:
            return 0.0
        
        # 纯度加权
        total_weight = sum(s.purity for s in seeds)
        avg_purity = total_weight / len(seeds)
        
        # 高纯度种子比例
        high_purity_ratio = sum(1 for s in seeds if s.purity >= 0.8) / len(seeds)
        
        # 数量因子（边际递减）
        count_factor = math.log(1 + len(seeds)) / math.log(11)  # 最大约 1
        
        # 组合
        potential = avg_purity * 0.4 + high_purity_ratio * 0.4 + count_factor * 0.2
        
        return potential
```

#### 3.2.3 阶段跃迁检测

```python
class PhaseTransitionDetector:
    """
    阶段跃迁检测器
    
    检测系统是否接近觉醒跃迁的临界点
    """
    
    # 临界区域的阈值
    CRITICAL_THRESHOLD = 0.85  # 势能超过此值认为接近临界
    
    @staticmethod
    def detect(
        awakening_energy: float,
        awakening_potential: float,
        emergence_history: List[Emergence],
        recent_strengths: List[float]
    ) -> Tuple[bool, str, float]:
        """
        检测是否处于跃迁临界状态
        
        Returns:
            (是否临界, 状态描述, 跃迁概率)
        """
        # 综合指标
        combined = awakening_energy * 0.4 + awakening_potential * 0.6
        
        # 最近涌现强度趋势
        strength_trend = 0.0
        if len(recent_strengths) >= 3:
            # 上升趋势
            if all(recent_strengths[i] <= recent_strengths[i+1] 
                   for i in range(len(recent_strengths)-1)):
                strength_trend = 0.2
        
        # 跃迁概率
        transition_probability = combined + strength_trend
        
        # 状态判断
        if combined >= PhaseTransitionDetector.CRITICAL_THRESHOLD:
            if strength_trend > 0:
                return True, "即将跃迁", transition_probability
            else:
                return True, "临界稳定", transition_probability
        elif combined >= 0.6:
            return False, "势能积累中", transition_probability
        else:
            return False, "稳定期", transition_probability
```
