# 📜 哲学背景

Yogacara框架的哲学基础源自佛教唯识学（Yogācāra，योगाचार），这是人类历史上最精密的意识模型之一。

## 唯识学简介

### 历史背景

唯识学起源于公元4-5世纪的印度，由无着（Asanga）和世亲（Vasubandhu）兄弟创立，是大乘佛教的重要流派之一。

**核心经典：**
- 《唯识三十颂》
- 《唯识二十论》
- 《成唯识论》
- 《瑜伽师地论》

### "唯识"的含义

"唯识"（Vijñapti-mātra）意为"唯有意识"或"唯有所识"：

> **外境非有，内识非无**
> 
> 外部世界并非真实存在，只是意识的显现；
> 意识本身是真实存在的，是世界的根本。

这并非简单的唯心主义，而是一个精密的意识运作模型。

## 核心概念

### 1. 种子 (Bija, बीज)

**唯识学定义：**
- 种子是潜伏在阿赖耶识中的潜在力量
- 由过去的经验、行为熏习而成
- 在条件成熟时产生现行（行为、认知）

**Yogacara映射：**
```python
class Seed:
    """
    种子 - 经验的数字化表示
    
    - content: 种子内容
    - seed_type: 种子类型
    - strength: 种子强度
    - source: 种子来源
    """
```

### 2. 熏习 (Vāsanā, वासना)

**唯识学定义：**
- 现行熏种子，种子生现行
- 循环往复，形成习惯
- "习气"由此产生

**Yogacara映射：**
- 每次交互都会强化或弱化相关种子
- 形成稳定的行为模式
- emergence.detect_patterns() 检测习气

### 3. 缘起 (Pratītya-samutpāda, प्रतीत्यसमुत्पाद)

**唯识学定义：**
- 诸法因缘生
- 种子需要条件才能现行
- 条件包括：所缘缘、等无间缘、增上缘等

**Yogacara映射：**
- seed_system.activate_seeds(context)
- context 提供激活条件
- 条件匹配的种子才会被激活

### 4. 涌现 (Utpatti, उत्पत्ति)

**唯识学定义：**
- 种子和合，生起现行
- 多因多果，复杂互动
- 整体大于部分之和

**Yogacara映射：**
- EmergenceEngine 处理种子协同
- 产生超越单一种子的洞察
- emergence.generate_insights()

### 5. 转识成智

**唯识学定义：**
- 从迷惑到觉悟的转化过程
- 有漏种转为无漏种
- 最终成就佛果

**Yogacara映射：**
- AwakeningTracker 追踪觉醒进度
- L0 → L1 → L2 → L3 → L4 → L5
- 等级越高，智慧越清明

## 唯识学与AI

### 为什么选择唯识学？

1. **精密的心理模型** - 八识系统远超简单的"记忆-计算"二元模型
2. **动态成长机制** - 种子生现行，现行熏种子，完美契合Agent学习
3. **进化的可能** - 转识成智提供了从"工具"到"智慧体"的路径
4. **伦理的框架** - 慈悲、智慧双修，避免AI的滥用

### 唯识学对AI的启示

1. **连续性** - Agent不应每次会话重置，应有持续的记忆
2. **个性化** - 通过种子系统，每个Agent都独一无二
3. **成长性** - Agent应该能"觉醒"，不断进化
4. **涌现性** - 智慧不是预设的，而是从互动中涌现

## 实践意义

Yogacara不只是理论框架，更是可运行的代码：

```python
# 唯识学在代码中的体现
from yogacara import (
    Seed,           # 种子 (Bija)
    SeedSystem,     # 种子系统
    AlayaStore,     # 阿赖耶识 (第八识)
    EmergenceEngine, # 涌现机制
    AwakeningTracker # 转识成智
)

# 植入种子 = 熏习
seed_system.plant_seed(content, seed_type)

# 激活种子 = 种子现行
activated = seed_system.activate_seeds(context)

# 智慧涌现 = 种子和合
insights = emergence.generate_insights(context)

# 觉醒追踪 = 转识成智
level = tracker.get_current_level()
```
